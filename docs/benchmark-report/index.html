Key Findings

All AI platforms produce code that compiles. None produce code that is fully correct.

Real-world AI-generated applications score an average health of 40/100 with 21 critical bugs across 4 codebases. The gap between "it compiles" and "it works" grows with complexity.

Critical Self-refine is useless

Asking an LLM to "try again" improves pass rates by only 0.6-1.2 percentage points on HumanEval. Platforms using this approach are wasting compute on the illusion of improvement.

Critical LLM-as-judge degrades quality

After 5 iterations, LLM-based code review drops pass rates from 99.4% to 97.2%. False positives cause the model to "fix" correct code. Using AI to review AI makes things worse.

Proven Formal verification converges to 100%

LUCID is the only verification approach that converges monotonically. HumanEval: 100% at k=3. SWE-bench: +36% improvement. Live coding: +26% improvement. No regressions.

Novel Verification improves model training

LUCID-verified data as DPO preference signal improves StarCoder2-3B by +2.0pp. The same data hurts as supervised fine-tuning (-4.0%). Verification is a contrastive training signal.

The Solution

LUCID: 3-Layer Formal Verification Pipeline

Unlike linters or type checkers, LUCID verifies semantic correctness. Does the code do what it claims? Does it meet the specification? Are there security holes?

L1: Extract

Identify testable claims from source code. What does this function promise? What invariants should hold? What contracts exist between components?

L2: Verify

Verify each claim against the source using formal analysis. Deterministic oracle — no probability, no hallucination, no regression with iteration.

L3: Remediate

Generate specific, actionable fix plans for each verification failure. Not "something is wrong" but "here is exactly what to change and why."

Why formal verification, not more AI? Our benchmark proves that using an LLM to judge LLM output degrades quality after 5 iterations (99.4% → 97.2%). Formal verification is deterministic: it either confirms or refutes a claim. It never hallucinates. It never regresses.
